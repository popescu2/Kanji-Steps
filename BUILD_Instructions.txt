Build Instructions for Kanji-Steps.

Background: This project was developed on Android Studio, and automatically comes with a Gradle build. As of now, in order to build this project, Android Studio is recommended.

Instructions: 

1. Clone repository into local directory.
2. Open Android Studio.
3. Select File>Open and select that directory.
4. Android Studio will automatically start configuring the Gradle Build.
5. Wait for it to finish. This may take several minutes.
